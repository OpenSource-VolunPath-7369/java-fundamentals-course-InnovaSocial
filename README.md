## Herramientas Utilizadas

Todas las prácticas del curso se realizan mediante plataformas 100% online:

- **Replit**: permite compilar y ejecutar código Java sin configuraciones previas.  
- **JDoodle**: útil para pruebas rápidas de fragmentos de código.  
- **OnlineGDB**: entorno online para ejecutar código más extenso con estructura de proyecto, estamos utilizando para este proyecto.

El estudiante no necesita realizar ninguna instalación en su computadora.  
Todo el curso está enfocado en reducir la barrera de entrada al aprendizaje de la programación.
