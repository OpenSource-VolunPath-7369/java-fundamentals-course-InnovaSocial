# Fundamentos de Java para Principiantes

## Resumen del Curso
Este curso de aproximadamente 1 hora introduce a estudiantes de secundaria a los fundamentos esenciales de la programación utilizando Java. Todas las prácticas y demostraciones se realizaron exclusivamente en OnlineGDB para facilitar el aprendizaje sin instalaciones.

El contenido avanza desde conceptos básicos hasta un mini proyecto final basado en Programación Orientada a Objetos (POO).

## Duración total
Aproximadamente 60 minutos distribuidos en 11 lecciones breves.

## Público objetivo
Estudiantes entre 12 y 17 años sin experiencia previa en programación.

## Herramientas necesarias
- Navegador web moderno  
- OnlineGDB: https://onlinegdb.com/  
(No requiere instalación local)

## Repositorio de Código Fuente
GitHub – java-fundamentals-course-InnovaSocial
